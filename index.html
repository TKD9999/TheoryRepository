수학 개념 정리

1. 함수 (Function)

한 집합의 원소를 다른 집합의 원소에 대응시키는 관계를 의미합니다.

y = f(x) : x는 정의역, y는 공역

2. 미분 (Differentiation)

함수의 순간변화율을 구하는 과정입니다.

f'(x) = lim[h→0] (f(x+h) - f(x)) / h

3. 적분 (Integration)

미분의 역과정으로, 곡선 아래의 면적을 구하는 과정입니다.

∫f(x)dx = F(x) + C

4. 삼각함수 (Trigonometric Functions)

직각삼각형의 각과 변의 비율을 나타내는 함수입니다.

sin θ = 대변/빗변
cos θ = 인접변/빗변
tan θ = 대변/인접변

5. 로그 (Logarithm)

지수함수의 역함수입니다.

logax = y ⇔ ay = x
